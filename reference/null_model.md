# Compute intercept-only model for regression models

This function computes the null-model (i.e. `(y ~ 1)`) of a model. For
mixed models, the null-model takes random effects into account.

## Usage

``` r
null_model(model, ...)

# Default S3 method
null_model(model, verbose = TRUE, ...)
```

## Arguments

- model:

  A (mixed effects) model.

- ...:

  Arguments passed to or from other methods.

- verbose:

  Toggle off warnings.

## Value

The null-model of `x`
